## 📖 Sobre o Repositório

Este repositório foi criado para documentar minha jornada de aprendizado no curso **Data Science com Power BI para Business Intelligence**.

Ao longo deste curso serão desenvolvidos projetos práticos, exercícios e estudos relacionados à análise de dados, modelagem de dados, Business Intelligence e criação de dashboards profissionais utilizando o Power BI.

---

# 🎯 Objetivos

- Aprender conceitos fundamentais de Business Intelligence.
- Trabalhar com processos ETL.
- Realizar tratamento e transformação de dados.
- Construir modelos de dados eficientes.
- Desenvolver métricas utilizando DAX.
- Criar dashboards profissionais.
- Aplicar técnicas de análise de dados para tomada de decisão.
- Construir um portfólio sólido de projetos em Power BI.
